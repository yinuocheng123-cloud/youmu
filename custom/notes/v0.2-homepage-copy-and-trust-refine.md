# 版本记录：v0.2-homepage-copy-and-trust-refine

## 1. 本轮修改目标
本轮目标是在 V0.1 静态首页 demo 基础上做文案、栏目语义、企业微信承接、SEO 与截图验证打磨。

重点是把首页从“可信商家”概念升级为“推荐厂商”，让表达更适合柚木产业从业者、工厂、品牌方和供应链伙伴，同时继续保持“柚木知识平台 + 好物推荐 + 推荐厂商展示 + 企业微信社群承接入口”的定位。

## 2. 实际修改文件
- `index.html`：调整导航、Hero、推荐厂商区、信任机制区、企业微信权益、SEO 信息和二维码替换注释。
- `styles.css`：新增《柚木选购避坑清单》轻量钩子按钮样式，并补充真实二维码图片替换样式。
- `custom/notes/v0.2-homepage-copy-and-trust-refine.md`：新增本轮记录。

## 3. “可信商家”改为“推荐厂商”的范围
本轮不是机械替换所有“商家”二字，而是围绕栏目概念做升级：

- 顶部导航由“可信商家”改为“推荐厂商”。
- Hero 说明中的“可信商家”改为“推荐厂商”。
- Hero 按钮“申请商家合作”改为“申请厂商合作”。
- 第六部分注释和栏目语义改为“推荐厂商区”。
- 推荐厂商区标题改为“让优质柚木厂商，被更多人看见。”
- 推荐厂商区说明改为“柚喜饰界不做简单广告展示，而是通过产品、案例、工艺、交付和用户反馈，帮助优质柚木厂商建立长期信任。”
- 信任机制标题改为“我们如何推荐柚木厂商？”
- 企业微信权益“商家合作申请”改为“厂商合作申请”。

保留了厂商卡片内“查看厂商故事”和“查看真实案例”，因为它们符合内容共创展示逻辑。

## 4. 企业微信二维码区是否保留
