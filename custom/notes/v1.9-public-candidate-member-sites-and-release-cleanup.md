# 版本记录：v1.9-public-candidate-member-sites-and-release-cleanup

## 1. 本轮目标

本轮目标是把站点推进到“可公开预览、可人工验收”的状态，重点完成三件事：

- 围绕五类柚木好物方向，新增一批公开资料候选品牌 / 企业会员站样板。
- 清理公开页面和公开数据源里的占位电话、占位邮箱和备案占位，不伪造真实资料。
- 复核移动端导航、二级栏目索引、会员站逻辑、风险边界、sitemap 和发布预检。

本轮属于：

- 内容补充
- 发布前清理
- 风险表达收口
- 可维护性提升

## 2. 背景与上下文

当前基础是 V1.8.3 已完成移动端导航全站可用与普通用户咨询入口下线。普通用户承接已经回到“社群交流 / 添加柚喜顾问”，厂商申请表单继续保留为“申请建立好物分类会员站”。

V1.8.3 后仍存在两个发布前问题：

- `check-release-readiness.mjs` 仍提示公开页面或数据源中有 `400-888-XXXX`、`hello@yuxishi.com`、`备案信息待补` 等占位项。
- 会员站体系虽然已有样板页，但庭院户外、茶室会客和更多真实品牌资料仍缺少公开候选结构，人工验收时不容易判断下一步从哪里补资料。

## 3. 公开候选品牌资料整理方式

本轮通过联网查找公开资料，优先使用品牌官网或品牌官网产品页，整理 11 个候选条目。

整理规则：

- 只使用公开资料改写摘要。
- 不复制官网大段原文。
- 不使用品牌 Logo、产品图或案例图。
- 不写成已入驻、认证、平台担保或正式会员。
- 每条都保留“是否联系确认：否”“是否图片授权：否”“是否为正式会员：否”的边界。

候选分类覆盖：

- 柚木地板：Indoteak Design、Arc Wood & Timbers。
- 柚木整装 / 木作配套：Noblewood Reclaimed Teak、Anthology Woods、WoodCo。
- 庭院户外柚木：Westminster Teak、Barlow Tyrie、Gloster。
- 茶室会客柚木：Hati Home、Teak Two。
- 柚木家具好物：Chic Teak、Teak Two。

## 8. 公开占位电话、邮箱、备案清理

本轮清理：

- `index.html` 中的 `400-888-XXXX`。
- `index.html` 中的 `hello@yuxishi.com`。
- `data/site-content.js` 中的 `400-888-XXXX`。
- `data/site-content.js` 中的 `hello@yuxishi.com`。
- `data/site-content.js` 中的 `备案信息待补`。

处理方式：

- 普通用户：改为“请通过社群交流区添加柚喜顾问”。
- 品牌 / 企业：改为“请通过会员站申请页整理资料后继续沟通”。
- 备案：改为“正式绑定自有域名前，将按实际情况补充备案信息。”

本轮没有伪造真实电话、真实邮箱或真实备案号。
