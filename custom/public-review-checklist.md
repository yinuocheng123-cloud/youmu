# 人工验收清单

文件说明：本文件用于正式发布前的人工作业验收。  
功能说明：把站点最关键的人工检查项整理成可勾选清单，便于上线前逐项确认。  

结构概览：
  第一部分：首页验收
  第二部分：知识页验收
  第三部分：方案页验收
  第四部分：会员站验收
  第五部分：案例页验收
  第六部分：表单页验收
  第七部分：技术验收

## 1. 首页验收

- [ ] 首页能正常打开
- [ ] 主导航只有五项入口
- [ ] 手机端菜单能正常打开和关闭
- [ ] 首页 CTA 没有误导到错误入口
- [ ] 首页没有交易承诺
- [ ] 首页 Hero 图、Logo、二维码图显示正常
- [ ] 首页“推荐厂商”模块已改成按五类好物分类查看会员站
- [ ] 首页候选品牌资料入口不会被误解为正式会员名单
- [ ] 庭院户外、茶室会客在首页明确为“待补充会员站样板”
- [ ] Footer 联系方式与二维码展示符合当前公开策略

## 2. 知识页验收

- [ ] 中文显示正常
- [ ] 页面不是骨架页
- [ ] 页面有来源说明或来源绑定提醒
- [ ] 页面没有夸大表达
- [ ] 页面没有把知识说明写成认证结论

## 3. 方案页验收

- [ ] 柚木地板表达清楚
- [ ] 页面不承诺真实交付
- [ ] 页面不写平台担保
- [ ] 方案页相关会员站入口已改成好物分类口径
- [ ] 图片、说明、边界提示一致

## 4. 会员站验收

- [ ] 页面表达为“按好物分类展示会员站”，不是简单推荐名单
- [ ] 五类好物分类说明完整
- [ ] 三个现有样板页已归入正确分类
- [ ] 庭院户外、茶室会客仅作为待补充样板出现
- [ ] 五项确认提示完整
- [ ] 会员展示不等于平台担保
- [ ] 样板页不会被误解为真实入驻
- [ ] 总入口页、样板页、申请页的边界口径一致
- [ ] 总入口页已补充温和的展示规则说明
- [ ] `vendors/candidates/index.html` 明确为公开资料候选品牌 / 企业会员站样板
- [ ] 候选资料不写成已入驻、认证、平台担保或正式会员
- [ ] 候选页未使用未授权品牌 Logo、产品图或案例图
- [ ] 候选资料来源已记录到 `custom/public-candidate-brand-source-log.md`

## 5. 案例页验收

- [ ] 明确属于样板案例
- [ ] 不写成功案例
- [ ] 不写真实成交案例

## 6. 表单页验收

- [ ] 好物分类会员站申请摘要能正常生成
- [ ] 静态表单说明清楚
- [ ] 没有提交成功或后台承诺
- [ ] 承接方式提示与真实流程一致

## 7. 技术验收

- [ ] 全站链接检查通过
- [ ] 编码检查通过
