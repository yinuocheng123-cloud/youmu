# 柚喜饰界官网首页预览确认包

## 1. 当前版本说明
这是“柚喜饰界官网首页预览版”，用于确认首页方向、视觉风格、Logo 方向和企业微信承接路径。

当前版本不是最终上线版，页面中的部分资料、图片、联系方式和二维码仍需要在后续阶段统一替换与确认。

## 2. 当前网站定位
柚喜饰界当前定位为：

柚木知识平台＋好物方案＋推荐厂商展示＋企业微信社群承接入口。

网站不按普通家具商城来做，不以购物车、价格和立即购买为核心，而是先帮助用户看懂柚木、建立信任、连接好物方案与推荐厂商，再通过企业微信进入后续咨询和社群承接。

## 3. 本次请甲方重点确认
- 是否认可“喜字方向 Logo”作为当前品牌方向。
- 是否认可首页高级、温润、低电商感的视觉风格。
- 是否认可网站不按家具商城，而按柚木生活方式与信任平台来做。
- 是否认可企业微信社群作为核心承接方式。
- 是否认可下一步进入真实素材替换。

## 4. 当前仍为示例内容
- 推荐厂商仍为样板展示位，不代表真实厂商资料或合作关系。
- 好物方案内容仍为示例推荐场景，柚木整装和柚木地板当前为未来承接方向表达，不代表真实交付承诺。
- 部分图片仍为占位视觉，后续需要替换为真实授权图片。
- 联系电话、邮箱、地址仍需项目方最终确认。
- 公众号二维码和企业微信二维码上线前仍需确认是否为最终投放版本。

## 5. 下一步流程
确认方向 → 提供真实资料 → 替换真实内容 → 上线前检查 → 部署上线。

## 6. GitHub Pages 公网预览说明
当前项目已进入 GitHub Pages 公网预览适配阶段，建议预览地址为：

`https://yinuocheng123-cloud.github.io/youmu/`

该地址用于甲方确认首页整体效果，不代表正式上线版本。正式上线前仍需确认图片授权、真实厂商资料、最终二维码、联系方式、备案信息和是否允许搜索引擎收录。

如 GitHub 仓库页面暂时找不到 Pages 开启入口，请查看 `custom/github-pages-guide.md`；发布目录应选择 `/ (root)`，表示仓库根目录。

如果分支发布方式保存后公网地址仍返回 404，V1.3 已补充 GitHub Actions Pages 部署方案。仓库推送后请到 `Actions` 页面查看 `Deploy static site to GitHub Pages` 是否运行成功，并在 `Settings -> Pages` 中确认 Source 为 `GitHub Actions`。

V1.4 在不改变首页核心模块顺序的前提下，重点优化了首屏视觉、移动端导航、好物方案分类和企业微信承接区。

本次请重点复核：
- 首屏是否比旧版更容易看清品牌定位和主文案。
- 手机端菜单是否清楚，是否方便进入柚木知识、好物方案、推荐厂商和社群入口。
- 好物方案分类是否能突出“柚木整装”和“柚木地板”两个重点方向，同时保持低电商感。
- 企业微信二维码区是否足够清楚，是否适合作为当前核心承接入口。

当前版本仍不代表最终上线内容。真实厂商、真实案例、真实资质、最终二维码、联系方式和图片授权仍需上线前统一复核。
