# 好物分类会员站推荐规则说明

文件说明：本文件用于沉淀“好物分类会员站”的内部运营规则。  
功能说明：帮助后续在不引入后台排序系统的前提下，先统一前台展示逻辑与内部判断标准。  

结构概览：
  第一部分：推荐体系定位
  第二部分：会员级别
  第三部分：推荐参考维度
  第四部分：资料完整度说明
  第五部分：展示边界
  第六部分：前台可公开说明

## 1. 推荐体系定位

“推荐厂商”不是平台担保，不是认证排名，也不是“谁交钱多谁排前面”。  
当前更稳妥的定位是：根据用户关注的柚木好物方向，结合会员级别、资料完整度、好物匹配度、内容活跃度、服务区域和合规状态，优先展示更适合被了解的品牌或企业会员站。

对用户而言，先看到的是：

- 柚木地板
- 柚木整装
- 庭院户外柚木
- 茶室会客柚木
- 柚木家具好物

对内部而言，再决定每个分类里哪些会员站更适合放在前面、哪些只保留样板或待补充。

## 2. 会员级别

建议使用安全表达：

- 基础展示会员
- 深度展示会员
- 共创展示会员

不使用以下表达：

- 认证会员
- 官方推荐会员
- 权威会员

说明：

- 会员级别用于区分内容深度与共创程度，不代表平台担保。

## 3. 推荐参考维度

分类展示时，可综合参考以下维度：

- 会员级别
- 资料完整度
- 好物分类匹配度
- 内容活跃度
- 服务区域
- 合规与授权状态

建议判断顺序：

1. 先看是否属于正确分类。
2. 再看五项确认是否完整。
3. 再看资料更新是否及时。
4. 最后结合服务区域和内容质量决定展示顺序。

## 4. 资料完整度说明

至少应确认以下内容：

- 企业资料
- 产品图片授权
- 联系方式确认
- 案例材料确认
- 服务边界确认
- 主营方向
- 服务区域

资料不完整时的处理建议：

- 可保留申请状态或待补充状态。
- 不要硬上真实会员站。
- 不要用模糊措辞包装成“已经完善”。

## 5. 展示边界

必须长期保留以下边界：

- 会员展示不等于平台担保。
- 平台不替代用户完成合同判断、产品验收、交付确认和售后责任确认。
- 平台不写成认证排名。
- 平台不写成交易承诺。

如果某会员站资料不完整：

- 可以保留分类说明。
- 可以保留申请入口。
- 可以保留待补充样板提示。
- 不要伪造真实品牌页、真实案例页或真实入驻状态。

## 6. 前台可公开说明

前台可以使用以下温和表达：

“本页会员站展示会综合参考会员级别、资料完整度、内容更新、好物分类匹配度和服务区域等因素。会员展示不等于平台担保，真实合作前仍需用户自行核实产品、合同、交付和售后服务。”

前台不建议使用以下表达：

- 官方推荐
- 平台认证
- 排名第一
- 放心购买
- 交易担保
