# 柚喜饰界预览说明

文件说明：本文件用于向项目方说明当前站点的预览定位与发布前边界。  
功能说明：帮助阅读者快速理解网站现阶段已经完成什么、还缺什么，以及为什么当前仍不应被视为正式上线站。  

结构概览：
  第一部分：当前版本定位
  第二部分：当前已完成内容
  第三部分：当前仍未完成内容
  第四部分：V1.8.1 新增说明

## 1. 当前版本定位

当前网站仍是“柚喜饰界”静态内容型预览站。  
它用于确认以下内容：

- 首页结构和视觉方向
- 五项一级导航结构
- 柚木知识、好物方案、按好物分类展示的会员站和社群承接的内容路径
- 正式发布前需要补齐的资料与人工验收流程

当前网站不是：

- 商城
- 后台系统
- 登录系统
- 支付系统
- 数据库驱动站点
- 平台担保型交易站

## 2. 当前已完成内容

当前版本已经完成：

- 首页整体结构
- 一级导航与移动端菜单
- 柚木知识页基础内容
- 好物方案页基础内容
- 按好物分类展示会员站的总入口与三个样板页
- 咨询摘要页与会员站申请摘要页
- 链接、编码、内容深度等基础检查脚本
- “推荐厂商”向“按好物分类展示会员站”的页面内核调整
- 正式发布前资料清单、人工验收清单、来源绑定清单、会员站资料模板和发布预检脚本

## 3. 当前仍未完成内容

当前仍未完成、且不能假装已经完成的内容包括：

- 最终企业微信二维码确认
- 最终公众号二维码确认
- 真实联系电话确认
- 真实联系邮箱确认
- 备案信息确认
- 真实品牌 / 企业会员站资料确认
- 真实案例资料确认
- 图片授权确认
- 逐页来源绑定确认
- 庭院户外会员站样板
- 茶室会客会员站样板

说明：

- 这些内容没有确认前，当前站点只适合作为预览站和上线前整理底稿。
- 没有真实资料时，不应把样板页写成真实入驻、真实案例或平台担保内容。

## 4. V1.8.1 新增说明

V1.8.1 的重点不是扩功能，而是把“推荐厂商 / 厂商会员站”进一步升级为“按柚木好物分类展示品牌或企业会员站”，并补齐内部推荐规则说明。当前已新增或更新：

- 首页厂商模块的五类好物会员站入口
- `vendors/index.html` 的五类入口与温和展示规则说明
- `forms/vendor-apply.html` 的好物分类会员站申请口径
- `custom/vendor-member-site-material-template.md`
- `custom/vendor-member-recommendation-rules.md`

当前结论：

- 网站结构已经基本稳定。
- 但距离正式发布仍差“真实资料补齐”和“人工复核闭环”。
- 当前更适合继续按清单补资料，而不是继续扩页面。
